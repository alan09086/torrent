# Benchmark v0.66.0: torrent vs rqbit vs qbittorrent

## Goal

5-trial benchmark comparing torrent v0.66.0, rqbit v8.1.1, and qbittorrent-nox v5.1.4 on the current Arch Linux ISO magnet (~1.5 GiB). Track wall time, CPU time (user+system), download speed, and peak RSS.

## Metrics

| Metric | Source |
|--------|--------|
| Wall time (s) | `/usr/bin/time -v` → "Elapsed (wall clock)" |
| CPU time (s) | `/usr/bin/time -v` → "User time" + "System time" |
| Download speed (MB/s) | `downloaded_bytes / wall_time` |
| Peak RSS (MiB) | `/usr/bin/time -v` → "Maximum resident set size" |
