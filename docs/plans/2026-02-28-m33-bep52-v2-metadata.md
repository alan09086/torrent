**Goal:** Parse, validate, and represent BitTorrent v2 torrents (BEP 52) — SHA-256 hashing, per-file Merkle hash trees, the v2 file tree format, v2 info-hash computation, and v2 magnet links.

**Architecture:** All new types live in `ferrite-core` (the leaf crate for core BitTorrent types). A new `sha2` dependency provides SHA-256. The existing `Magnet` struct extends to handle dual v1/v2 info-hashes. The `ferrite` facade crate re-exports all new public types.

**Tech Stack:** Rust 2024, `sha2` crate (SHA-256), `serde`/`serde_bytes`, `ferrite-bencode` (BencodeValue for file tree parsing), `data-encoding` (base32 for multihash magnets), `thiserror`.

## Senior Review: Key Architectural Decisions

1. **Separate `TorrentMetaV2` from `TorrentMetaV1`** — v2 has a fundamentally different info dict structure (file tree vs flat file list, no `pieces` field, `meta version` key). A unified enum (`TorrentMeta::V1(..)|V2(..)`) would add indirection for every consumer. Keeping them separate matches how libtorrent and rqbit handle this. Hybrid torrents (M35) will compose both.

2. **Parse file tree via `BencodeValue` walk, not serde** — The v2 file tree uses arbitrary filenames as dict keys with a magic `""` key for file attributes. This isn't representable with `#[derive(Deserialize)]`. Instead, deserialize as `BencodeValue::Dict` and walk recursively.

3. **Merkle tree as flat `Vec<Id32>`** — Standard binary heap layout (children of node `i` at `2i+1`, `2i+2`). Efficient for both construction and layer extraction. No allocations beyond the initial vec.

4. **`sha256()` as public function in `ferrite-core`** — Mirrors the existing `sha1()` pattern. Consumers import `ferrite_core::sha256` (or `ferrite::core::sha256`).

5. **Multihash for v2 magnets** — BEP 52 uses the multihash format for `urn:btmh:` params: `0x1220` prefix + 32 SHA-256 bytes = 68 hex chars. Implemented inline (no multihash crate needed for a single codec).

```rust
/// Encode as base32 string (no padding), used in magnet links.
pub fn to_base32(&self) -> String {
    data_encoding::BASE32_NOPAD.encode(&self.0)
}

/// Create from a base32 string (52 chars).
pub fn from_base32(s: &str) -> Result<Self, Error> {
    let bytes = data_encoding::BASE32_NOPAD
        .decode(s.as_bytes())
        .map_err(|e| Error::InvalidHex(format!("base32: {e}")))?;
    Self::from_bytes(&bytes)
}

/// Encode as a multihash hex string (BEP 52 magnet `urn:btmh:` format).
///
/// Format: `0x12` (SHA-256 function code) + `0x20` (32-byte length) + hash.
/// Result is 68 hex characters.
pub fn to_multihash_hex(&self) -> String {
    let mut buf = Vec::with_capacity(34);
    buf.push(0x12); // SHA-256 function code
    buf.push(0x20); // 32-byte digest length
    buf.extend_from_slice(&self.0);
    hex::encode(buf)
}

/// Create from a multihash hex string (68 chars).
///
/// Validates the SHA-256 function code (`0x12`) and length prefix (`0x20`).
pub fn from_multihash_hex(s: &str) -> Result<Self, Error> {
    let bytes = hex::decode(s).map_err(|e| Error::InvalidHex(e.to_string()))?;
    if bytes.len() != 34 {
        return Err(Error::InvalidHashLength {
            expected: 34,
            got: bytes.len(),
        });
    }
    if bytes[0] != 0x12 {
        return Err(Error::InvalidHex(format!(
            "multihash function code 0x{:02x}, expected 0x12 (SHA-256)",
            bytes[0]
        )));
    }
    if bytes[1] != 0x20 {
        return Err(Error::InvalidHex(format!(
            "multihash digest length 0x{:02x}, expected 0x20 (32)",
            bytes[1]
        )));
    }
    Self::from_bytes(&bytes[2..])
}
```

This implements the BEP 52 per-file Merkle hash tree. Key properties:
- Leaf nodes are SHA-256 hashes of 16 KiB blocks
- Tree is padded to a power of 2 with zero hashes
- Parent = SHA-256(left_child || right_child)
- "Piece layer" is the tree level where each hash covers `piece_length` bytes
- Stored as a flat vec in binary heap order (root at index 0)

```rust
//!
//! Each file in a v2 torrent has its own Merkle tree built from 16 KiB block

///
/// Stored as a flat array in standard binary heap layout:
/// - Root at index 0
/// - Children of node `i` at `2*i + 1` and `2*i + 2`
/// - Leaves occupy the last `leaf_count` positions
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct MerkleTree {
    /// All nodes from root to leaves.
    nodes: Vec<Id32>,
    /// Number of leaf positions (always a power of 2, includes padding).
    leaf_count: usize,
}
```

```rust
impl MerkleTree {
    /// Construct a Merkle tree from leaf hashes.
    ///
    /// Pads the leaf count to the next power of 2 with `Id32::ZERO`.
    /// Each parent is `SHA-256(left_child || right_child)`.
    pub fn from_leaves(leaves: &[Id32]) -> Self {
        if leaves.is_empty() {
            return MerkleTree {
                nodes: vec![],
                leaf_count: 0,
            };
        }

        // Pad to next power of 2
        let leaf_count = leaves.len().next_power_of_two();
        let total_nodes = 2 * leaf_count - 1;
        let mut nodes = vec![Id32::ZERO; total_nodes];

        // Fill leaf positions (last leaf_count slots)
        let leaf_start = leaf_count - 1;
        for (i, hash) in leaves.iter().enumerate() {
            nodes[leaf_start + i] = *hash;
        }
        // Remaining leaf slots are already Id32::ZERO (padding)

        // Build tree bottom-up
        for i in (0..leaf_start).rev() {
            let left = nodes[2 * i + 1];
            let right = nodes[2 * i + 2];
            nodes[i] = hash_pair(left, right);
        }

        MerkleTree { nodes, leaf_count }
    }

    /// The root hash of the tree.
    pub fn root(&self) -> Id32 {
        if self.nodes.is_empty() {
            Id32::ZERO
        } else {
            self.nodes[0]
        }
    }

    /// Number of leaf positions (padded to power of 2).
    pub fn leaf_count(&self) -> usize {
        self.leaf_count
    }

    /// Total depth of the tree (root = depth 0, leaves = depth `tree_depth()`).
    pub fn depth(&self) -> usize {
        if self.leaf_count == 0 {
            return 0;
        }
        self.leaf_count.trailing_zeros() as usize
    }

    /// Get all hashes at a given depth (0 = root, `depth()` = leaves).
    ///
    /// Returns an empty slice if depth exceeds tree depth.
    pub fn layer(&self, depth: usize) -> &[Id32] {
        let tree_depth = self.depth();
        if depth > tree_depth || self.nodes.is_empty() {
            return &[];
        }
        let start = (1 << depth) - 1; // 2^depth - 1
        let count = 1 << depth; // 2^depth
        &self.nodes[start..start + count]
    }

    /// Extract the piece layer — the level where each hash covers `blocks_per_piece` leaves.
    ///
    /// `blocks_per_piece` = `piece_length / 16384` (the number of 16 KiB blocks per piece).
    /// The piece layer is at depth `log2(leaf_count / blocks_per_piece)` from the root,
    /// or equivalently `log2(leaf_count) - log2(blocks_per_piece)` from the root.
    ///
    /// Returns all leaves if `blocks_per_piece >= leaf_count` (single-piece file).
    pub fn piece_layer(&self, blocks_per_piece: usize) -> &[Id32] {
        if self.leaf_count == 0 || blocks_per_piece == 0 {
            return &[];
        }
        if blocks_per_piece >= self.leaf_count {
            // File fits in one piece — root IS the piece hash
            return self.layer(0);
        }
        let num_pieces = self.leaf_count / blocks_per_piece;
        let piece_depth = (num_pieces.trailing_zeros()) as usize;
        // piece_depth from root: log2(num_pieces)
        // But layer() counts from root, so depth = tree_depth - log2(blocks_per_piece)
        let depth = self.depth() - blocks_per_piece.trailing_zeros() as usize;
        self.layer(depth)
    }

    /// Compute the root hash from a list of hashes by building a temporary tree.
    ///
    /// Useful for verifying piece layer → root without storing the full tree.
    pub fn root_from_hashes(hashes: &[Id32]) -> Id32 {
        if hashes.is_empty() {
            return Id32::ZERO;
        }
        if hashes.len() == 1 {
            return hashes[0];
        }
        let tree = MerkleTree::from_leaves(hashes);
        tree.root()
    }

    /// Get the leaf hashes (the last level of the tree).
    pub fn leaves(&self) -> &[Id32] {
        self.layer(self.depth())
    }

    /// Access all nodes in the tree (binary heap layout).
    pub fn nodes(&self) -> &[Id32] {
        &self.nodes
    }
}

/// Compute the parent hash from two children: `SHA-256(left || right)`.
fn hash_pair(left: Id32, right: Id32) -> Id32 {
    let mut buf = [0u8; 64];
    buf[..32].copy_from_slice(left.as_bytes());
    buf[32..].copy_from_slice(right.as_bytes());
    crate::sha256(&buf)
}
```

```rust
//! v2 represents files as nested dictionaries where path components are dict
//! keys and the empty-string key `""` holds file attributes. This cannot be

use std::collections::BTreeMap;

/// Properties of a single file in the v2 file tree.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct V2FileAttr {

}
```

```rust
impl FileTreeNode {
    /// Parse a v2 file tree from a `BencodeValue::Dict`.
    ///
    /// The dict is walked recursively. At each level, if a key `""` exists,
    /// the node is a file (the `""` value contains `length` and optionally
    /// `pieces root`). Otherwise, each key is a path component leading to a
    /// child node.
    pub fn from_bencode(value: &ferrite_bencode::BencodeValue) -> Result<Self, Error> {
        let dict = match value {
            ferrite_bencode::BencodeValue::Dict(d) => d,
            _ => return Err(Error::InvalidTorrent("file tree node must be a dict".into())),
        };

        // Check for the special "" key (file attributes)
        if let Some(attrs_val) = dict.get(b"".as_slice()) {
            // This is a file node
            let attr = parse_file_attrs(attrs_val)?;
            return Ok(FileTreeNode::File(attr));
        }

        // Otherwise it's a directory
        let mut children = BTreeMap::new();
        for (key_bytes, child_val) in dict {
            let key = String::from_utf8(key_bytes.clone())
                .map_err(|_| Error::InvalidTorrent("file tree key is not valid UTF-8".into()))?;
            let child = FileTreeNode::from_bencode(child_val)?;
            children.insert(key, child);
        }

        Ok(FileTreeNode::Directory(children))
    }

    /// Flatten the tree into a list of files with full paths.
    pub fn flatten(&self) -> Vec<V2FileInfo> {
        let mut result = Vec::new();
        self.flatten_into(&mut Vec::new(), &mut result);
        result
    }

    fn flatten_into(&self, prefix: &mut Vec<String>, result: &mut Vec<V2FileInfo>) {
        match self {
            FileTreeNode::File(attr) => {
                result.push(V2FileInfo {
                    path: prefix.clone(),
                    attr: attr.clone(),
                });
            }
            FileTreeNode::Directory(children) => {
                for (name, child) in children {
                    prefix.push(name.clone());
                    child.flatten_into(prefix, result);
                    prefix.pop();
                }
            }
        }
    }
}

/// Parse file attributes from the `""` dict entry.
fn parse_file_attrs(value: &ferrite_bencode::BencodeValue) -> Result<V2FileAttr, Error> {
    let dict = match value {
        ferrite_bencode::BencodeValue::Dict(d) => d,
        _ => return Err(Error::InvalidTorrent("file attributes must be a dict".into())),
    };

    let length = match dict.get(b"length".as_slice()) {
        Some(ferrite_bencode::BencodeValue::Integer(n)) => {
            if *n < 0 {
                return Err(Error::InvalidTorrent("file length cannot be negative".into()));
            }
            *n as u64
        }
        _ => return Err(Error::InvalidTorrent("file missing 'length' key".into())),
    };

    let pieces_root = match dict.get(b"pieces root".as_slice()) {
        Some(ferrite_bencode::BencodeValue::Bytes(b)) => Some(
            Id32::from_bytes(b)
                .map_err(|_| Error::InvalidTorrent("pieces root must be 32 bytes".into()))?,
        ),
        None => None,
        _ => return Err(Error::InvalidTorrent("pieces root must be bytes".into())),
    };

    Ok(V2FileAttr {
        length,
        pieces_root,
    })
}
```

```rust
impl InfoDictV2 {
    /// Flatten the file tree into a list of files with paths.
    pub fn files(&self) -> Vec<V2FileInfo> {
        self.file_tree.flatten()
    }

    /// Total size of all files in bytes.
    pub fn total_length(&self) -> u64 {
        self.files().iter().map(|f| f.attr.length).sum()
    }

    /// Number of pieces across all files.
    ///
    /// In v2, each file is aligned to piece boundaries — pieces never span files.
    /// The total piece count is the sum of per-file piece counts.
    pub fn num_pieces(&self) -> u32 {
        self.files()
            .iter()
            .filter(|f| f.attr.length > 0)
            .map(|f| f.attr.length.div_ceil(self.piece_length) as u32)
            .sum()
    }

    /// Compute per-file piece counts and cumulative piece index offsets.
    ///
    /// Returns a list of `(file_info, piece_offset, piece_count)` for each
    /// non-empty file. The `piece_offset` is the global piece index where
    /// this file's pieces start.
    pub fn file_piece_ranges(&self) -> Vec<(V2FileInfo, u32, u32)> {
        let mut offset = 0u32;
        let mut ranges = Vec::new();
        for file in self.files() {
            if file.attr.length == 0 {
                continue;
            }
            let count = file.attr.length.div_ceil(self.piece_length) as u32;
            ranges.push((file, offset, count));
            offset += count;
        }
        ranges
    }
}

    }

    if info.piece_length < 16384 {
        return Err(Error::InvalidTorrent(format!(

        return Err(Error::InvalidTorrent(format!(
            "v2 piece length must be a power of 2, got {}",
            info.piece_length
        )));
    }

    Ok(())
}
```

```rust
/// Parse a v2 .torrent file from raw bytes (BEP 52).
///
/// Computes the info-hash by finding the raw byte span of the "info" key
/// and SHA-256-hashing it directly (not the re-serialized form).
pub fn torrent_v2_from_bytes(data: &[u8]) -> Result<TorrentMetaV2, Error> {
    // Step 1: Compute info hash from raw bytes (SHA-256)
    let info_span = ferrite_bencode::find_dict_key_span(data, "info")?;
    let info_hash = crate::sha256(&data[info_span]);

    // Truncated hash for v1-compatible tracker announces
    let mut truncated = [0u8; 20];
    truncated.copy_from_slice(&info_hash.as_bytes()[..20]);
    let info_hash_truncated = Id20(truncated);

    // Step 2: Deserialize top-level structure as BencodeValue
    let top: ferrite_bencode::BencodeValue = ferrite_bencode::from_bytes(data)?;
    let top_dict = match &top {
        ferrite_bencode::BencodeValue::Dict(d) => d,
        _ => return Err(Error::InvalidTorrent("top-level must be a dict".into())),
    };

    // Step 3: Parse the info dict
    let info_val = top_dict
        .get(b"info".as_slice())
        .ok_or_else(|| Error::InvalidTorrent("missing 'info' key".into()))?;

    let info_dict = match info_val {
        ferrite_bencode::BencodeValue::Dict(d) => d,
        _ => return Err(Error::InvalidTorrent("'info' must be a dict".into())),
    };

    let name = extract_string(info_dict, b"name")?;
    let piece_length = extract_integer(info_dict, b"piece length")? as u64;
    let meta_version = extract_integer(info_dict, b"meta version")? as u64;

fn parse_piece_layers(
    dict: &BTreeMap<Vec<u8>, ferrite_bencode::BencodeValue>,
) -> Result<BTreeMap<Id32, Vec<u8>>, Error> {
    let layers_val = match dict.get(b"piece layers".as_slice()) {
        Some(v) => v,
        None => return Ok(BTreeMap::new()),
    };

    let layers_dict = match layers_val {
        ferrite_bencode::BencodeValue::Dict(d) => d,
        _ => return Err(Error::InvalidTorrent("'piece layers' must be a dict".into())),
    };

    let mut result = BTreeMap::new();
    for (key_bytes, value) in layers_dict {
        let root = Id32::from_bytes(key_bytes)
            .map_err(|_| Error::InvalidTorrent("piece layers key must be 32 bytes".into()))?;
        let hashes = match value {
            ferrite_bencode::BencodeValue::Bytes(b) => b.clone(),
            _ => {
                return Err(Error::InvalidTorrent(
                    "piece layers value must be bytes".into(),
                ))
            }
        };
        if !hashes.is_empty() && hashes.len() % 32 != 0 {
            return Err(Error::InvalidTorrent(format!(
                "piece layer length {} is not a multiple of 32",
                hashes.len()
            )));
        }
        result.insert(root, hashes);
    }

    Ok(result)
}
```

```rust
impl TorrentMetaV2 {
    /// Validate that piece layers are consistent with the file tree.
    ///
    /// For each file larger than `piece_length`:
    /// - A piece layer entry must exist keyed by the file's `pieces_root`
    /// - The layer must contain exactly `ceil(file_length / piece_length)` hashes
    ///   (with trailing hashes for padding beyond file end omitted per BEP 52)
    ///
    /// Files ≤ `piece_length` do not need piece layer entries.
    pub fn validate_piece_layers(&self) -> Result<(), Error> {
        for file in self.info.files() {
            let length = file.attr.length;
            if length == 0 {
                continue;
            }

            let pieces_root = match &file.attr.pieces_root {
                Some(r) => r,
                None => continue,
            };

            if length <= self.info.piece_length {
                // File fits in one piece — no piece layer needed
                continue;
            }

            // File is larger than piece_length — piece layer required
            let layer = self.piece_layers.get(pieces_root).ok_or_else(|| {
                Error::InvalidTorrent(format!(
                    "missing piece layer for file '{}' (pieces_root: {})",
                    file.path.join("/"),
                    pieces_root.to_hex()
                ))
            })?;

            let expected_pieces = length.div_ceil(self.info.piece_length) as usize;
            let actual_hashes = layer.len() / 32;

            if actual_hashes != expected_pieces {
                return Err(Error::InvalidTorrent(format!(
                    "piece layer for '{}' has {} hashes, expected {} (file {} bytes, piece {} bytes)",
                    file.path.join("/"),
                    actual_hashes,
                    expected_pieces,
                    length,
                    self.info.piece_length
                )));
            }
        }

        Ok(())
    }

    /// Get the piece layer hashes for a specific file.
    ///
    /// Returns `None` if the file fits in a single piece (no layer needed)
    /// or if no layer exists. Each hash is 32 bytes (SHA-256).
    pub fn file_piece_hashes(&self, pieces_root: &Id32) -> Option<Vec<Id32>> {
        let layer = self.piece_layers.get(pieces_root)?;
        let count = layer.len() / 32;
        let mut hashes = Vec::with_capacity(count);
        for i in 0..count {
            let start = i * 32;
            let mut arr = [0u8; 32];
            arr.copy_from_slice(&layer[start..start + 32]);
            hashes.push(Id32(arr));
        }
        Some(hashes)
    }
}
```

```rust
use url::Url;

use crate::error::Error;
use crate::hash::{Id20, Id32};

/// Parsed magnet link (BEP 9 + BEP 52).
///
/// Supports v1 (`urn:btih:`), v2 (`urn:btmh:`), and hybrid magnets.
///
/// Format: `magnet:?xt=urn:btih:<v1-hash>&xt=urn:btmh:<v2-multihash>&dn=<name>&tr=<tracker>`
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct Magnet {
    /// v1 info hash (SHA-1, 20 bytes). `None` for v2-only magnets.
    pub info_hash_v1: Option<Id20>,
    /// v2 info hash (SHA-256, 32 bytes). `None` for v1-only magnets.
    pub info_hash_v2: Option<Id32>,
    /// Display name (optional).
    pub display_name: Option<String>,
    /// Tracker URLs.
    pub trackers: Vec<String>,
    /// Peer addresses (x.pe parameter).
    pub peers: Vec<String>,
}

impl Magnet {
    /// Get the v1 info hash, for backward compatibility.
    ///
    /// # Panics
    /// Panics if this is a v2-only magnet with no v1 hash.
    pub fn info_hash(&self) -> Id20 {
        self.info_hash_v1
            .expect("called info_hash() on a v2-only magnet; use info_hash_v1 or info_hash_v2")
    }

    /// Whether this magnet has a v2 info hash.
    pub fn is_v2(&self) -> bool {
        self.info_hash_v2.is_some()
    }

    /// Whether this is a hybrid magnet (both v1 and v2 hashes).
    pub fn is_hybrid(&self) -> bool {
        self.info_hash_v1.is_some() && self.info_hash_v2.is_some()
    }

    /// Parse a magnet URI string.
    pub fn parse(uri: &str) -> Result<Self, Error> {
        let normalized = if let Some(rest) = uri.strip_prefix("magnet:?") {
            format!("magnet://dummy?{rest}")
        } else {
            return Err(Error::InvalidMagnet("must start with 'magnet:?'".into()));
        };

        let url = Url::parse(&normalized)
            .map_err(|e| Error::InvalidMagnet(format!("URL parse error: {e}")))?;

        let mut info_hash_v1 = None;
        let mut info_hash_v2 = None;
        let mut display_name = None;
        let mut trackers = Vec::new();
        let mut peers = Vec::new();

        for (key, value) in url.query_pairs() {
            match key.as_ref() {
                "xt" => {
                    if let Some(hash_str) = value.strip_prefix("urn:btih:") {
                        info_hash_v1 = Some(parse_v1_info_hash(hash_str)?);
                    } else if let Some(hash_str) = value.strip_prefix("urn:btmh:") {
                        info_hash_v2 = Some(parse_v2_info_hash(hash_str)?);
                    }
                }
                "dn" => {
                    display_name = Some(value.into_owned());
                }
                "tr" => {
                    trackers.push(value.into_owned());
                }
                "x.pe" => {
                    peers.push(value.into_owned());
                }
                _ => {} // Ignore unknown parameters
            }
        }

        if info_hash_v1.is_none() && info_hash_v2.is_none() {
            return Err(Error::InvalidMagnet(
                "missing xt=urn:btih: or xt=urn:btmh:".into(),
            ));
        }

        Ok(Magnet {
            info_hash_v1,
            info_hash_v2,
            display_name,
            trackers,
            peers,
        })
    }

    /// Convert back to a magnet URI string.
    pub fn to_uri(&self) -> String {
        let mut parts = Vec::new();

        if let Some(ref v1) = self.info_hash_v1 {
            parts.push(format!("xt=urn:btih:{}", v1.to_hex()));
        }
        if let Some(ref v2) = self.info_hash_v2 {
            parts.push(format!("xt=urn:btmh:{}", v2.to_multihash_hex()));
        }

        if let Some(ref name) = self.display_name {
            parts.push(format!(
                "dn={}",
                url::form_urlencoded::byte_serialize(name.as_bytes()).collect::<String>()
            ));
        }

        for tracker in &self.trackers {
            parts.push(format!(
                "tr={}",
                url::form_urlencoded::byte_serialize(tracker.as_bytes()).collect::<String>()
            ));
        }

        format!("magnet:?{}", parts.join("&"))
    }
}

/// Parse a v1 info hash from hex (40 chars) or base32 (32 chars).
fn parse_v1_info_hash(s: &str) -> Result<Id20, Error> {
    match s.len() {
        40 => Id20::from_hex(s),
        32 => Id20::from_base32(&s.to_ascii_uppercase()),
        _ => Err(Error::InvalidMagnet(format!(
            "v1 info hash must be 40 hex or 32 base32 chars, got {} chars",
            s.len()
        ))),
    }
}

/// Parse a v2 info hash from multihash hex (68 chars).
fn parse_v2_info_hash(s: &str) -> Result<Id32, Error> {
    Id32::from_multihash_hex(s)
        .map_err(|e| Error::InvalidMagnet(format!("v2 multihash: {e}")))
}
```
