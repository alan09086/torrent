# M18: ferrite-utp — BEP 29 uTP Protocol

## Overview

Standalone uTP (Micro Transport Protocol) implementation over UDP with LEDBAT congestion control. Session integration is deferred to M19.

## Architecture

### Actor Model

Single `SocketActor` owns the UDP socket and all connections. Incoming packets are dispatched by `(addr, connection_id)`. Connection state machine produces `Vec<ConnAction>` (Send/Deliver/Closed) — never touches I/O directly.

```
UtpSocket (cloneable handle)
    |
SocketActor (owns UdpSocket, tokio::select! loop)
    |--- Connection (conn_id=1) --- UtpStream
    |--- Connection (conn_id=2) --- UtpStream
    |--- Connection (conn_id=3) --- UtpStream
```

### Crate Structure

| File | Role |
|------|------|
| `seq.rs` | `SeqNr(u16)` wrapping arithmetic |
| `packet.rs` | BEP 29 header + SACK extension encode/decode |
| `congestion.rs` | LEDBAT controller |
| `conn.rs` | Connection state machine (pure logic, no I/O) |
| `stream.rs` | `AsyncRead + AsyncWrite` bridge via mpsc channels |
| `listener.rs` | Incoming connection acceptor |
| `socket.rs` | Actor owning UDP socket + dispatch |

### Key Design Decisions

- **Single actor, no shared socket**: Matches DhtActor pattern, no concurrent send contention
- **Action-based connection I/O**: Testable without real UDP
- **mpsc channels for Stream↔Actor**: `poll_recv()` integrates with `AsyncRead`/`AsyncWrite`
- **50ms tick interval**: Simpler than per-connection timers; min RTO is 500ms
- **Send buffering**: Data exceeding congestion window is buffered and flushed on ACK

### LEDBAT Parameters

- Target delay: 100ms
- Base delay: rolling min over 2 minutes
- Window adjustment: `cwnd += MAX_CWND_INCREASE * (off_target/target) * (bytes_acked/cwnd)`
- Loss: `cwnd = max(cwnd/2, MIN_WINDOW)`
- RTT: Jacobson EWMA, min RTO 500ms, initial 1000ms

### Connection ID Assignment (BEP 29)

- Initiator: random `recv_id`, `send_id = recv_id + 1`
- Responder: `recv_id = initiator_recv_id + 1`, `send_id = initiator_recv_id`

### Public API

```rust
let (socket, listener) = UtpSocket::bind(UtpConfig::default()).await?;
let stream = socket.connect(addr).await?;           // outbound
let (stream, addr) = listener.accept().await?;       // inbound
// UtpStream: AsyncRead + AsyncWrite + Unpin + Send
```
